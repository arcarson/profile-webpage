Academic Procession Application Web App

This is an internal web app for the University of Melbourne created to simplify and streamline the process for academics applying to be part of academic processions for graduation ceremonies and for the hiring of necessary regalia. It was also created to automate much of the backend of this process which had been very manually intensive.

The app was created in Ruby on Rails and hosted on Heroku.
